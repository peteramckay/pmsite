Introducing Beams

A secure, fast web framework

Beams is an open, full-stack framework for building websites, tailored for a future in which the entire web is served using HTTPS encryption.

The major browser makers, including Mozilla and Google, have become adamant that such a transition needs to happen in response to rising user concerns over privacy. Google has also begun favoring HTTPS sites in search results. So the pressure on every publisher on the web, big or small, is mounting.

Beams is built to make the switch to HTTPS easier. Its core is an Nginx script that handles front-end HTTPS validation, reverse-proxies to your content, and maintains performance throughout.

You can download the current version of Beams, which is in public beta release, from us here. Or, if you'd like to participate in development of future releases of Beams, please visit its page on GitHub, our chief venue for open-source collaboration.

Beams is (and always will be) free to use under MIT license. We're adding features all the time, with feedback and contributions from the community.
